# Analisis-Rotacion-Personal-Power-BI
Dashboard interactivo en Power BI para analizar la rotación laboral. Incluye métricas clave (empleados, renuncias, rotación, salario y satisfacción), junto con visualizaciones sobre antigüedad, departamentos y horas extra. El repositorio contiene el archivo PBIX, las medidas DAX y el dataset utilizado.
